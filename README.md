# 🔍 GitHub RAG (Retrieval-Augmented Generation)

> ⚠️ **This project is still under active development. Features and structure are subject to change.**

---

## 📌 Overview

A RAG-based pipeline that lets you query GitHub repositories using natural language. Indexes code, docs, and files from a repo and retrieves relevant context to answer questions powered by an LLM.

---

## 🚧 Status

This project is in early development. Core functionality is being built out — expect breaking changes and incomplete features.

---

## 🛠️ Tech Stack

- **LangChain** — RAG pipeline orchestration
- **PineCode DB** — Vector store for embeddings
- **OpenAI/ Mistral** — LLM + Embeddings
- **GitHub API/ temp files** — Repository ingestion
- **Python 3.10+**
